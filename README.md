Jogo desenvolvido para a 6ª Edição da Game Jaaj

Loop e um jogo de Puzzle, onde o objetivo do jogador e mover os objetos da sala afim de encontrar a saida, porém se o tempo acabar os objetos voltaram a sua posição inicial.

O jogo foi desenvolvido com a intenção de adquirir conhecimento e experiência no motor de jogos Game Maker Studio, todo o jogo foi desenvolvido dentro do prazo de 1 semana, uma das regras estipuladas para participar da Game Jam.

Link do jogo no Itch.io: https://bladedragonx.itch.io/clocker
Link da Game Jam: http://jams.gamejolt.io/gamejaaj6
